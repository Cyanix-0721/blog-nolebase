# Paru

## 1 Install

```bash
sudo pacman -S paru
```

## 2 Usage

* -S：安装软件包

* -U：更新软件包

* -R：移除软件包

* -Syu：同步软件包列表并更新所有已安装的软件包

* -Syyu：同步软件包列表并更新所有已安装的软件包，以及它们的依赖项

* -Q：查询软件包信息

* -Qi：显示软件包的详细信息

* -Ss：搜索软件包

* -P：打印软件包的依赖关系树

* -G：显示软件包的依赖关系图

* -A：安装软件包及其所有依赖项

* -D：移除软件包及其所有依赖项

* --noconfirm：在安装、更新或移除软件包时不询问用户确认

* --force：强制安装、更新或移除软件包，即使存在冲突

* --skip-broken：在安装、更新或移除软件包时跳过损坏的软件
